# 診断ルーブリック — スコアリング基準

本ドキュメントは、物語分析モードにおける4カテゴリ x 5段階の評価基準を定義する。
スコアリング時はフォーマット別重み付け表（format-guides.md）を参照して加重平均を算出する。

---

## 評価スケール

| グレード | 意味 | 数値換算 |
|---------|------|---------|
| **A** | 優秀 — プロレベルの完成度 | 4.0 |
| **B** | 良好 — 機能しているが改善余地あり | 3.0 |
| **C** | 要改善 — 明確な問題がある | 2.0 |
| **D** | 不十分 — 大幅な修正が必要 | 1.0 |
| **F** | 機能していない — 根本的な再設計が必要 | 0.0 |

---

## カテゴリ1: 構造スコア（Structure）

対象フォーマットのビート構造（映画: STC15ビート、ドラマ: 8ビート、ショートドラマ: 4ビート、マンガ: 起承転結）に対して、各ビートが適切な位置に存在するかを評価する。

### 評価基準

| グレード | 基準 |
|---------|------|
| **A** | フォーマット所定のビートの90%以上が適切な位置（基準値 ±5%以内）に存在する。三幕（または相当する区分）の比率が基準通り。全体の流れに論理的な破綻がない |
| **B** | ビートの70-89%が適切な位置に存在。一部のビートに軽微な位置ズレ（±10%以内）があるが、物語の骨格は機能している |
| **C** | ビートの50-69%が確認できる。主要ビート（Catalyst, Midpoint, All Is Lost相当）のうち1つが欠落 or 大きくズレている。構造を意識はしているが精度が不足 |
| **D** | ビートの30-49%のみ確認できる。主要ビートが2つ以上欠落。幕の比率が大きく崩れている（例: 第一幕が40%以上） |
| **F** | ビートの30%未満。構造がほぼ存在しない。場面の羅列に近い状態 |

### チェックポイント

- [ ] フォーマット所定の全ビートを洗い出し、存在の有無を確認
- [ ] 各ビートの位置（%）を計測し、基準値との乖離を算出
- [ ] 三幕（または区分）の比率を計測
- [ ] ビート間の因果関係が成立しているか

---

## カテゴリ2: 感情アーク明度（Arc Clarity）

物語全体の感情曲線が、6つの基本型（Man in a Hole / Cinderella / Oedipus / Icarus / Rags to Riches / Tragedy）のいずれかに明確に沿っているかを評価する。

### 評価基準

| グレード | 基準 |
|---------|------|
| **A** | 6型のいずれか1つに明確に一致する。各ビートでの感情値が理想曲線から大きく逸脱しない（乖離スコア15%以内）。読者/視聴者が「この物語はこういう話だ」と一言で言える |
| **B** | 型は明確に識別可能だが、一部のビート（1-2箇所）で理想曲線からのズレがある（乖離スコア15-30%）。例: Man in a Holeなのに谷が浅い、Cinderellaなのに最終上昇が弱い |
| **C** | 2つの型の混合が見られる。方向性は見えるが、どちらの型なのか判然としない。もしくは型は識別できるが、乖離が30-50%。感情の流れに「?」を感じる箇所が複数ある |
| **D** | アークが不明瞭。感情の方向性が途中で何度も変わり、一貫したパターンが読み取れない。どの型にも明確にフィットしない |
| **F** | 感情の起伏がほぼない（フラットライン）、またはランダムに上下する。物語としての感情体験が成立していない |

### チェックポイント

- [ ] 主要ビートごとの感情値（高/中/低）をプロットする
- [ ] 最も近い型を特定し、理想曲線との乖離を評価
- [ ] 感情の転換点がストーリー上の転換点と一致しているか
- [ ] 主人公の内面変化と感情曲線が連動しているか

---

## カテゴリ3: ペーシング（Pacing）

フォーマット別のビートタイミング基準（format-guides.md参照）に対して、実際のビート配置がどの程度合致しているかを評価する。

### 評価基準

| グレード | 基準 |
|---------|------|
| **A** | 全ビートが基準タイミングの ±5%以内に収まっている。テンポに違和感がなく、「長い」「退屈」と感じる区間がない。テンションの緩急が意図的に設計されている |
| **B** | 大半のビートが ±10%以内。1-2箇所で軽微なテンポの乱れがある（例: Set-Upがやや長い）が、全体としては快適なテンポ |
| **C** | 複数のビートが ±15%以上ズレている。明確に「間延び」または「駆け足」と感じる区間がある。例: Fun and Gamesが短すぎてプレミスを楽しめない |
| **D** | ビートの半数以上が ±20%以上ズレている。テンポが全体的に遅い or 速すぎる。緩急のコントロールが効いていない |
| **F** | ビートの配置にタイミングの概念がほぼない。第一幕だけで全体の50%を使う、クライマックスが唐突に来る等、ペーシングが破綻している |

### フォーマット別の特記事項

**映画:**
- Catalystが15%以降 → ペーシング減点の主因になりやすい
- Fun and Gamesが短い → 観客がプレミスを楽しめない
- 第三幕が30%以上 → 冗長に感じる

**ドラマ:**
- 1話内のビート間隔が均等すぎると単調になる。緩急の波を確認
- エピソード前半にビートが集中し後半がダレるパターンに注意

**ショートドラマ:**
- Hookが3秒以内に来ないと致命的（最重要チェック項目）
- Escalationの密度が命。10秒以上「何も起きない」区間はNG
- 全体が1-3分のため、数秒のズレが大きな影響を持つ

**マンガ:**
- 起が6ページ以上 → 読者離脱リスク
- 転がページめくりと連動していない → 演出効果の損失
- 見開きの後にテンポが落ちすぎていないか

### チェックポイント

- [ ] 各ビートの実タイミング（%）を計測
- [ ] format-guides.mdの基準値との差分を一覧化
- [ ] ±5% / ±10% / ±15% / ±20%超のビート数をカウント
- [ ] 「間延び区間」「駆け足区間」を具体的にマーク

## カテゴリ4: 引き（Engagement）

視聴者・読者を物語に引き込み、離脱させない力を評価する。フォーマットごとに評価観点が異なる。

### 映画の引き

| グレード | 基準 |
|---------|------|
| **A** | Opening Imageが強烈で世界に引き込む。Climaxが感情的に満足できる。Final Imageが深い余韻を残し、Opening Imageと美しく呼応する |
| **B** | 上記3要素のうち2つが高水準。1つに改善余地あり |
| **C** | 3要素のうち1つのみ高水準。または全体的に「悪くないが印象に残らない」 |
| **D** | Opening Imageが弱く掴めない、かつClimax or Final Imageも弱い |
| **F** | 開始5分で興味を失う。クライマックスに到達しても感情が動かない。終わり方が投げやり |

### ドラマの引き

| グレード | 基準 |
|---------|------|
| **A** | 全エピソード末にレベル2以上のクリフハンガーがある。B Storyが感情深度を持ち、視聴者がキャラクターに感情移入している。「次も見なければ」と思わせる力が持続 |
| **B** | クリフハンガーは概ね機能しているが、1-2話で弱い。B Storyは存在するが感情的インパクトがやや不足 |
| **C** | クリフハンガーが形式的。B Storyがメインに貢献していない。「見てもいいし見なくてもいい」レベル |
| **D** | クリフハンガーが複数話で欠落。B Storyが不明瞭。継続視聴の動機が薄い |
| **F** | エピソード末がただ終わるだけ。サブプロットが機能していない。1話で離脱する可能性が高い |

### ショートドラマの引き

| グレード | 基準 |
|---------|------|
| **A** | 冒頭3秒で確実にスワイプを止める。毎話クリフハンガーで終わる。クリフハンガーの種類がローテーションされている。課金壁前のフック力が極めて高い |
| **B** | 冒頭フックは機能するが、3秒ルールを満たさない話が1-2話ある。クリフハンガーは毎話あるが種類が偏る |
| **C** | フックが弱い話が複数ある。クリフハンガーが形式的（「次回に続く」だけ等）。中盤で離脱リスクが高い |
| **D** | 冒頭が遅い話が半数以上。クリフハンガーが複数話で欠落。スクロールで飛ばされる可能性大 |
| **F** | スワイプ阻止力がない。クリフハンガーがない。視聴継続の仕組みが設計されていない |

### マンガの引き

| グレード | 基準 |
|---------|------|
| **A** | ページめくりの「転」が効果的に機能し驚きがある。見開きの衝撃度が高く配置も適切。話末の引きが毎話強力で「次話を読まずにはいられない」 |
| **B** | ページめくり演出は概ね機能。見開きは効果的だが配置にやや改善余地。話末の引きは大半の話で機能 |
| **C** | ページめくりの活用が不十分（転が右ページにない等）。見開きが多すぎる or 少なすぎる。話末の引きが弱い回がある |
| **D** | ページめくり演出がほぼ意識されていない。見開きの配置が不適切。話末に引きがない回が複数 |
| **F** | コマ割りが単調でページめくりの概念がない。見開きが効果的に使われていない。話末が投げやり |

### チェックポイント

- [ ] フォーマット別の評価観点を全てチェック
- [ ] 「最も引きが強い箇所」と「最も弱い箇所」を特定
- [ ] 引きの種類のバリエーションを確認（偏っていないか）

## 総合評価の算出

### 計算式

```
総合スコア = (構造スコア x 構造重み) + (感情アークスコア x 感情アーク重み) + (ペーシングスコア x ペーシング重み) + (引きスコア x 引き重み)
```

### フォーマット別重み付け（format-guides.mdより）

| 評価カテゴリ | 映画 | ドラマ | ショートドラマ | マンガ |
|------------|------|-------|-------------|-------|
| 構造 | 30% | 25% | 15% | 25% |
| 感情アーク | 30% | 25% | 15% | 20% |
| ペーシング | 20% | 25% | 35% | 25% |
| 引き | 20% | 25% | 35% | 30% |

### 数値換算とグレード変換

1. 各カテゴリのグレードを数値に変換（A=4.0, B=3.0, C=2.0, D=1.0, F=0.0）
2. フォーマット別重みで加重平均を算出
3. 加重平均を総合グレードに変換:

| 加重平均 | 総合グレード |
|---------|------------|
| 3.5-4.0 | **A** — 高い完成度。微調整で更に良くなる |
| 2.5-3.4 | **B** — 良好。特定箇所の改善で大きく向上する |
| 1.5-2.4 | **C** — 要改善。構造的な見直しが必要 |
| 0.5-1.4 | **D** — 大幅修正。骨格から再設計を検討 |
| 0.0-0.4 | **F** — 根本的な再構築が必要 |

### 出力フォーマット

```
## 診断結果

| カテゴリ | スコア | 重み | 加重値 |
|---------|-------|------|-------|
| 構造 | [A-F] | [%] | [値] |
| 感情アーク | [A-F] | [%] | [値] |
| ペーシング | [A-F] | [%] | [値] |
| 引き | [A-F] | [%] | [値] |
| **総合** | **[A-F]** | — | **[加重平均]** |
```

## 改善提案テンプレート

各カテゴリのスコアがC以下の場合、以下のテンプレートから該当する提案を出力する。

### 構造（Structure）がC以下の場合

| 問題 | 提案 |
|------|------|
| Catalystの欠落 | 「物語を動かすきっかけ（Catalyst）が見当たりません。主人公の日常を壊す決定的な事件を、全体の10-12%付近に配置してください」 |
| Catalystの位置が遅い | 「Catalystの位置が遅すぎます（現在: [X]%）。全体の10-12%付近（映画なら12-15分）に移動を検討してください」 |
| Midpointの欠落 | 「Midpointが不明瞭です。物語の50%付近にFalse Victory（偽りの勝利）またはFalse Defeat（偽りの敗北）を配置し、後半の方向を変えてください」 |
| All Is Lostの欠落 | 「最大の危機（All Is Lost）が見つかりません。75%付近に主人公が全てを失う瞬間を作ってください。"死の匂い"が必要です」 |
| 三幕比率の崩れ | 「第[X]幕が全体の[Y]%を占めており、基準（25/50/25）から大きくズレています。[具体的な調整提案]」 |
| Break into 2の欠落 | 「主人公が能動的に第二幕に踏み出す瞬間（Break into 2）がありません。25%付近に主人公自身の選択による転換を入れてください」 |
| Break into 3の欠落 | 「第三幕への転換（Break into 3）が不明瞭です。80%付近でAストーリーとBストーリーが融合し、解決の糸口を掴む瞬間を作ってください」 |

### 感情アーク（Arc Clarity）がC以下の場合

| 問題 | 提案 |
|------|------|
| 型が不明瞭 | 「感情アークが複数の型の間で揺れています。[候補型1]と[候補型2]のどちらを目指すか決めて、逸脱箇所を修正してください」 |
| 谷が浅い | 「All Is Lost〜Dark Night of the Soulでの感情の谷が浅いです。主人公がもっと深く落ちないと、最終的な上昇のカタルシスが弱くなります」 |
| 最終上昇が弱い | 「Climax〜Final Imageの感情上昇が弱いです。読者/視聴者が"報われた"と感じるために、感情の頂点をもっと高く設計してください」 |
| フラットライン | 「感情の起伏がほとんどありません。主人公にとっての"賭け金（Stakes）"を高くし、成功/失敗の振れ幅を大きくしてください」 |
| 内面変化の不一致 | 「物語上の転換点と主人公の感情が連動していません。外的事件が主人公の内面にどう影響するかを明示してください」 |

### ペーシング（Pacing）がC以下の場合

| 問題 | 提案 |
|------|------|
| Set-Upが長すぎる | 「セットアップが全体の[X]%を占めており長すぎます。日常の描写を圧縮し、Catalystを前倒ししてください」 |
| Fun and Gamesが短い | 「プレミスを楽しませる区間（Fun and Games）が短すぎます。この物語の"売り"をもっと展開するシーンを追加してください」 |
| 第三幕が冗長 | 「Finaleが長すぎます（現在: [X]%）。25%以内に収まるよう、クライマックス後の処理を簡潔にしてください」 |
| 全体的に遅い | 「テンポが全体的に遅いです。各シーンの目的を明確にし、目的を達成したらすぐに次のシーンに移ってください。"入りは遅く、出は早く"が基本です」 |
| 全体的に速すぎる | 「展開が急すぎて感情がついていけません。特に[具体箇所]の前後に"溜め"のシーンを入れ、感情の準備時間を作ってください」 |
| 冒頭が遅い（ショートドラマ） | 「冒頭3秒以内にフックがありません。最も衝撃的/魅力的な要素を冒頭に持ってきてください。説明は後回しで構いません」 |

### 引き（Engagement）がC以下の場合

| 問題 | 提案 |
|------|------|
| Opening Imageが弱い（映画） | 「冒頭の映像/場面に引き込む力がありません。物語のテーマを象徴する印象的なビジュアル/状況から始めてください」 |
| Final Imageの不在（映画） | 「Opening Imageと呼応するFinal Imageがありません。"変化前→変化後"を視覚的に対比させるラストカットを設計してください」 |
| クリフハンガーの欠落（ドラマ/ショートドラマ） | 「エピソード[X]の末尾にクリフハンガーがありません。疑問型/危機型/暴露型/反転型のいずれかで"続きが気になる"状態にしてください」 |
| クリフハンガーの偏り | 「クリフハンガーの種類が[型名]に偏っています。他の型も混ぜてマンネリを防いでください」 |
| ページめくり未活用（マンガ） | 「転のシーンが右ページ末に来ていません。コマの配置を調整し、ページをめくった瞬間に驚きが来るよう設計してください」 |
| 見開き過多/過少（マンガ） | 「見開きが[多すぎ/少なすぎ]ます。1話に0-2回を目安に、最もインパクトのある瞬間に限定してください」 |
| 話末の引きが弱い（マンガ） | 「最終ページの引きが弱いです。登場/宣言/危機/謎のいずれかのパターンで"次話を読まずにはいられない"状態を作ってください」 |
| B Storyの感情深度不足（ドラマ） | 「B Storyが形式的で感情に訴えていません。メインキャラクターの内面的な課題と結びつけ、テーマを深める役割を持たせてください」 |

### 提案の出力ルール

1. スコアがC以下のカテゴリのみ提案を出力する（A/Bは省略）
2. 各カテゴリにつき最大3つの提案に絞る（最も影響が大きい順）
3. 提案には必ず「現状の数値」と「目標の数値」を含める（計測可能な場合）
4. 抽象的なアドバイスではなく、具体的なアクションを提示する
5. フォーマット固有の提案を優先する（汎用提案は後回し）
